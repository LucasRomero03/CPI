# 1-ETAPA
- Entrar no site de twillo e se cadastrar. 
- ir no menu lateral após criação da conta.
    - Messaging -> try it out -> send a WPP message.
        - aqui vai ter um numero ficticio para você usar o código que aparece.

- Após isso bastar clicar para ele abrir no wpp e verificar enviando mensagem. 
- Depois ir em account dashboard e procurar seu token auth e o account SID , que serão usados posteriormente para validar no cpi.
